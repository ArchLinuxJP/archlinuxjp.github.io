シンプル、軽量なディストリビューション

Arch Linux のウェブサイトにようこそ。Keep It Simple を標榜する、軽量で柔軟性に優れた Linux® ディストリビューションです。 公式パッケージは i686 と x86-64 アーキテクチャに最適化されています。さらに公式パッケージを補足するものとしてコミュニティによって運営されているパッケージリポジトリがあり、日々、その質量ともに成長しています。 Arch Linux の強固なコミュニティは多様かつ助けになり、広範なスキルセットと、それによって Arch を利用していることに誇りを持っています。フォーラムやメーリングリストを見て参加してみて下さい。Slack で Arch ユーザーと会話することもできます (誰でも自由に参加できます)。また、Arch について知識を深めたいと考えたのであれば、ArchWiki も覗いてみましょう。
